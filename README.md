## Run Locally

Clone the project

```bash
  git clone https://github.com/shivam-0510/totalitycorp-frontend-challenge
```

Go to the project directory

```bash
  cd totalitycorp-frontend-challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
